# My-first-project
 
# 🧮 Умный калькулятор с тёмной темой и памятью

## 📖 Описание проекта
Это многофункциональный веб-калькулятор, который умеет не только складывать, вычитать, умножать и делить, но и запоминать результаты вычислений. Проект создан для демонстрации навыков работы с GitHub.

## ✨ Функциональность
1. ✅ Базовые арифметические операции (+, -, ×, ÷)
2. ✅ Обработка ошибки деления на ноль
3. ✅ Проверка корректности ввода чисел
4. ✅ Тёмная тема оформления (переключается одной кнопкой)
5. ✅ Светлая тема оформления
6. ✅ Плавные анимации при наведении на кнопки
7. ✅ Сохранение результатов в память (до 15+ значений)
8. ✅ Просмотр всей сохранённой истории
9. ✅ Восстановление последнего сохранённого результата
10. ✅ Использование любого результата из истории
11. ✅ Отображение времени сохранения каждого результата
12. ✅ Очистка всей памяти одной кнопкой
13. ✅ Очистка полей ввода и результата
14. ✅ Информативный дисплей с подсказками
15. ✅ Адаптивный дизайн под любые экраны
16. ✅ Визуальная индикация текущего действия
17. ✅ Сохранение результатов с меткой времени

## 🚀 Как пользоваться
1. Введите два числа в поля ввода
2. Нажмите кнопку операции (+, -, ×, ÷)
3. Результат появится внизу
4. Чтобы запомнить результат — нажмите «💾 Запомнить»
5. Чтобы посмотреть сохранённое — нажмите «📋 Вспомнить»
6. Любой результат из истории можно повторно использовать
7. Переключайте тему кнопкой вверху
