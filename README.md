# 🧮 Умный калькулятор

## 📖 Описание проекта
Это веб-приложение — многофункциональный калькулятор с современным дизайном. Он работает прямо в браузере, не требует установки и доступен по ссылке. Проект создан для демонстрации навыков работы с GitHub.

## ✨ Функциональность
Основные операции

1. Сложение (+) — складывает два числа
2. Вычитание (-) — вычитает второе число из первого
3. Умножение (×) — умножает два числа
4. Деление (÷) — делит первое число на второе

Дополнительные операции

1. Проценты (%) — вычисляет процент от числа (например, 10% от 200 = 20)
2. Возведение в квадрат (x²) — умножает число само на себя (5² = 25)
3. Квадратный корень (√) — извлекает квадратный корень из числа (√16 = 4)

Защита от ошибок

1. Проверка деления на ноль — при делении на 0 появляется сообщение «Ошибка: деление на ноль!»
2. Проверка пустых полей — если поля не заполнены, появляется подсказка «⌨️ Введите оба числа» (без красной ошибки)
3. Проверка отрицательного корня — корень из отрицательного числа выдаёт ошибку

Память результатов

1. Сохранение результата — кнопка «💾 Запомнить результат» добавляет текущий результат в список памяти
2. Время сохранения — каждый сохранённый результат сопровождается временем (часы:минуты:секунды)
3. Просмотр памяти — все сохранённые числа отображаются в виде списка с временем
4. Использование из памяти — кнопка «Использовать» подставляет сохранённое число обратно в калькулятор
5. Очистка памяти — кнопка «🗑️ Очистить всю память» удаляет все сохранённые результаты

Интерфейс и темы

1. 4 цветные темы — светлая ☀️, тёмная 🌙, синяя 💙, розовая 💗 (переключаются одной кнопкой)
2. Дисплей-индикатор — показывает текущее действие (например, 10 + 5 = 15)
3. Анимация кнопок — при наведении кнопки увеличиваются и меняют цвет
4. Адаптивный дизайн — корректно отображается на телефонах и компьютерах

Удобство использования

1. Кнопка «Очистить всё» — одной кнопкой очищает оба поля ввода и результат
2. Понятные подсказки — вместо ошибок пользователь видит вежливые сообщения
3. Работа без перезагрузки — все действия происходят мгновенно на одной странице

## 🎯 Цель создания
Демонстрация навыков разработки и работы с Git

🚀 Как пользоваться

1. Введите два числа в поля «Первое число» и «Второе число»
2. Нажмите на операцию:
   · + — сложение
   · - — вычитание
   · × — умножение
   · ÷ — деление
   · % — процент от второго числа
   · x² — возведение первого числа в квадрат
   · √ — квадратный корень из первого числа
3. Результат появится в розовом блоке внизу
4. Сохраните результат в память — кнопка «💾 Запомнить результат»
5. Используйте сохранённое — нажмите «Использовать» рядом с любым числом в списке
6. Смените тему — нажмите на иконку темы вверху (☀️, 🌙, 💙, 💗)
7. Очистите всё — кнопка «🗑️ Очистить всё»
